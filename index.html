About NeLang

NeLang is an open-source programming language utilizing Native Nepali keywords structurally mapped atop a lightning-fast Python Machine Bytecode compiler engine. It embraces Python's beautiful indentation-based formatting rules while injecting localized cultural terminologies.

Key Philosophy: Programming logic is universal, but syntax should feel close to home. NeLang exists to bridge modern language capabilities to Nepali syntax readers—perfect for localized logic mapping, education, and general-purpose development applications.

Installation / Usage

Install the compiled wheel from the source and execute your `.nl` files securely via the global NeLang executable hook.

pip install nelang

To run a script simply execute:

nelang run filename.nl

Core Syntax

NeLang embraces robust mapping over explicit data states. Every example executes natively across standard compilers.

Variables

Dynamically instantiated identical to Python standard conventions.

x = 10

Print

Outputs strictly to conventional `stdout` standard streams.

lekha("Hello")

Conditionals

Evaluate blocks natively evaluating against booleans.

yadi x > 5:
    lekha("Thulo")
tyasovaye:
    lekha("Sano")

Loops

Continuous iterative boundaries mapped to condition states.

jaba x < 5:
    lekha(x)

Functions

Spawn isolated executing closures explicitly mapping parameters environments.

karya greet():
    lekha("Namaste")

Return

Escapes active functional closures resolving an expression to its root caller.

firta value

Boolean

Literal semantic truth states.

satya, jhuto

Logical

Boolean mapping operators natively supporting AST chained sequences.

ra, wa, hoina

Lists

Standardized structural storage parsing.

nums = [1, 2, 3]

For Loop

Traverse iterable sets tracking object yields block by block.

lagi x vitra nums:
    lekha(x)

Input

Blocks the execution thread polling raw OS standard inputs parsing user strings.

name = lina("Enter name: ")

Comments

Lexer escapes strictly ignored throughout structural runtime.

# yo comment ho

karya compute_circle_area(radius):
    yadi radius <= 0:
        lekha("Invalid radius")
        firta 0
    tyasovaye:
        firta math.pi * math.pow(radius, 2)

r = 5
lekha("Radius 5 area:", compute_circle_area(r))
